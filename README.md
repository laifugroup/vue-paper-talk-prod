# 纸上谈兵(荣耀)

## 项目介绍
一个基于Vue 3和Vite开发的抖音发奖状pdf系统。基于jspdf生成pdf文件，并且使用pdf.js预览pdf文件。
* 系统运行还需要  [抓取抖音弹幕](https://gitee.com/apebyte/dy-barrage-grab)
