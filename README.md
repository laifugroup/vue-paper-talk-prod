# 纸上谈兵(荣耀)

## 项目介绍
一个基于Vue 3和Vite开发的纸上谈兵项目。

## 项目效果

![项目截图](/images/paper.png)

## 项目说明
- 项目基于Vue 3和Vite开发，使用了Vue 3的Composition API和TypeScript。

## 项目运行
```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
